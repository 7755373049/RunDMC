MarkLogic Server License Options

You can download MarkLogic Server and be up and running immediately with one of the following Standard Edition license options.

Trial License

The Trial License allows you to evaluate MarkLogic Server Standard Edition for free for 30 days. A Trial License enables full access to optional features including, geospatial support, entity enrichment, and alerting, for the duration of the trial period. The Trial License is limited to 40 gigabytes of content and can be run on a dual CPU server. The Trial License allows MarkLogic Server to be used for evaluation purposes only and not for production applications or the development of production applications. Support is provided informally through the developer network.

Community License

The Community License is available free of charge for personal non-commercial projects. It does not restrict you to a fixed trial period, but is limited to a single- or dual-CPU server and 10 gigabytes of content. Unlike the Trial License, the Community License does not include access to optional features, such as geospatial support, entity enrichment, and alerting. Use of MarkLogic Server under the Community License is restricted to two instances per organization.

Academic License

The Academic License is available to students and instructors at educational institutions. It provides a convenient way to take advantage of MarkLogic Server for class projects or instruction. Like the Trial License, it includes geo-spatial support, entity enrichment, and alerting and is limited to 40 gigabytes of content. However, the Academic License is free for instructional use for up to a year. For more information or to request an Academic License, please contact MarkLogic.
